Getting Started with HTML, CSS & JavaScript

This is a tiny interactive demo inspired by w3schools. Feel free to copy/paste or modify these snippets.

1. Simple HTML Button

<button onclick="alert('Hello, GrassOS!')">Click me!</button>

2. Basic CSS (Change button color on hover)


button {
  background-color: #7cb342; /* grassy green */
  color: white;
  border: none;
  padding: 10px 20px;
  cursor: pointer;
}
button:hover {
  background-color: #689f38; /* darker green */
}

3. JavaScript Snippet (Log to console)


<script>
  console.log('Welcome to GrassOS learning corner!');
</script>

Try copying these into your own files or live editor and see what happens!
